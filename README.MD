# 📊 Análise de Dados com SQL e Python

Projeto focado no desenvolvimento de habilidades em SQL e Python para análise de dados.

---

## 🎯 Objetivo

Praticar e consolidar conceitos de SQL aplicados à análise de dados, incluindo:

* SELECT, WHERE
* JOIN
* GROUP BY
* HAVING
* Subqueries
* CTE e Window Functions

---

## 🛠️ Tecnologias

* SQL (SQLite)
* Python
* Pandas
* SQLAlchemy
* Git e GitHub

---

## 📂 Estrutura

* Arquivos `.sql` com exercícios e práticas
* `etl.py` com pipeline de dados
* `etl_projeto.sql` com query utilizada

---

## ⚙️ Pipeline

O script Python realiza:

* leitura de query SQL
* execução no banco SQLite
* transformação com pandas
* gravação em tabela final
